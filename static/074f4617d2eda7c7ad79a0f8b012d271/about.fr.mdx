## Au sujet de ce cours

Les buts de ce cours sont :    
- de réaliser une RESTful API via Node.js & Express ;
- de développer un frontend interactif et moderne en JS / HTML / CSS ;
- de construire une Single-Page Application (SPA) intégrant un frontend moderne et une RESTful API ;
- d'obtenir les compétences pour développer un projet web qui vous tient à cœur !

Si vous souhaitez participer à la mise à jour du cours, n'hésitez pas à faire part de modifications en soumettant des Pull Requests ou des Issues sur le <a href="https://github.com/e-vinci/web2" target="_blank"> web repository de ce site web.</a>

Nous espérons que vous apprendrez en vous amusant !
